# Cognitive Modeler

A desktop application for modeling and analyzing cognitive maps.

## Description

Cognitive Modeler is a cross-platform Electron-based application for creating and analyzing cognitive maps - a visual tool for understanding complex systems and scenarios.

## Tech Stack

**Frontend:**

- Vue 3 + TypeScript
- Vite
- Element Plus (UI components)
- Cytoscape.js (graph visualization)
- ECharts (charts and diagrams)

**Backend:**

- Python 3.13+
- FastAPI
- NumPy

**Desktop:**

- Electron (Windows/Linux)
